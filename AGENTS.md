# AGENTS.md

AI coding agents: the org-wide contract in https://github.com/ajar-protocol/.github/blob/main/AGENTS.md applies fully here.

Repo-specific rules:

- Directory entries need a working link and one-line description.
- Do not rank, endorse, or editorialize beyond factual category placement.
- Compatibility claims need a conformance report once the harness lands.
- Keep private or not-yet-published repos as plain code text with the phase note.
- Preserve neutral criteria for closed-source, commercial, and open-source entries.
- Check links before changing a listing when network access is available.
